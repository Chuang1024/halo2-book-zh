# [译] The halo2 Book 中文版

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Status](https://img.shields.io/badge/status-completed-brightgreen.svg)]()

本项目是对 Zcash 团队编写的 [The halo2 Book](https://zcash.github.io/halo2/) 的全中文翻译，涵盖了从理论背景到算术化（Arithmetization）实现的全部内容。

---

## 📖 关于本项目

**halo2** 是目前零知识证明（ZKP）领域最先进的框架之一。为了降低中文开发者的学习门槛，本项目完成了对官方文档的完整汉化，并对书中配套的 Rust 代码进行了详细的中文注释。
